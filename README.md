# Native Graph

A high-performance graph topology library written in Rust, designed for speed, safety, and cross-platform compatibility.

## Features

- 🚀 **Blazingly Fast**: built in Rust for maximum performance

- 🔒 **Memory Safe**: leverages Rust's ownership model for thread-safe operations

- 🔗 **Multi-Language Bindings**: native support for
Python, WebAssembly (WASM), and Flutter

- 📊 **Flexible Topologies**: support for various graph structures and algorithms
